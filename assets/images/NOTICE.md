# Third-party icon assets

## Lucide icons

The following icons are derived from [Lucide](https://lucide.dev) and used under the
[ISC License](https://github.com/lucide-icons/lucide/blob/main/LICENSE). The ISC license
does not require runtime attribution; this file exists for audit purposes only.

| File | Lucide icon | Purpose |
| --- | --- | --- |
| `whatsapp_generic.svg` | `message-circle` | Generic chat icon for the WhatsApp contact link — deliberately not a reproduction of the WhatsApp logo. |
| `telegram_generic.svg` | `send` | Generic paper-plane/"send" icon for the Telegram contact link — deliberately not a reproduction of the Telegram logo. |
| `instagram_generic.svg` | `camera` | Generic camera icon for the Instagram contact link — deliberately not a reproduction of the Instagram logo. |
| `map_pin_generic.svg` | `map-pin` | Generic map pin used for the 2GIS map link — deliberately not a reproduction of the 2GIS logo. |

visually reproducing brand logo silhouettes (trademark risk for App Store / Play Market listings).

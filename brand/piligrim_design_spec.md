# PILIGRIM — Design Specification for App Development
> Основано на официальном брендбуке. Версия 1.0

---

## 1. О бренде

**Название:** PILIGRIM / Piligrim  
**Позиционирование:** Кухня свободы и традиций  
**Слоган (RU):** Кухня свободы и традиций  
**Слоган (KZ):** Дәстүрдің дәмі, еркіндік лебі  
**Слоган (EN):** Spirit and culture cuisine  
**Суть бренда:** Вкус жизни. Путь героя  
**Концепция:** Духовно-гастрономическое путешествие. Гость = герой. Персонал = проводники.

## 2. Цветовая палитра

Все цвета названы по-казахски — это часть идентичности бренда.

| Название | Казахское имя | HEX | Применение |
|---|---|---|---|
| Тёмно-коричневый / уголь | Қара жер (Чёрная земля) | `#3D3A38` | Основной фон (тёмная тема), фоны карточек |
| Стальной голубой | Мөлдір су (Прозрачная вода) | `#7BA5B8` | Главный акцентный цвет, иконки, логотип на тёмном фоне, highlights |
| Золотисто-песочный | Сары дала (Жёлтая степь) | `#C4956A` | Второстепенный акцент, CTA на светлом фоне, тёплые элементы |
| Тёмно-красный | Піскен жеміс (Спелый плод) | `#8B1A1A` | Акцент действия, кнопки завершения, спец. элементы |
| Кремово-белый | Ақ аспан (Белое небо) | `#F2EDE4` | Светлый фон, текст на тёмном, карточки на светлой теме |

### Расширенная палитра (производные)

```
--color-earth:       #3D3A38   /* Қара жер — тёмный фон */
--color-water:       #7BA5B8   /* Мөлдір су — акцент */
--color-steppe:      #C4956A   /* Сары дала — тёплый акцент */
--color-fruit:       #8B1A1A   /* Піскен жеміс — красный */
--color-sky:         #F2EDE4   /* Ақ аспан — светлый фон */

/* Производные */
--color-earth-deep:  #2A2826   /* Более глубокий фон, navbar */
--color-water-muted: #5A8FA8   /* Pressed state акцента */
--color-sky-warm:    #EDE7DC   /* Чуть теплее, второй светлый фон */
--color-text-light:  #F2EDE4   /* Текст на тёмном */
--color-text-dark:   #3D3A38   /* Текст на светлом */
--color-divider:     rgba(242,237,228,0.12) /* Разделители на тёмном */
```

### Правила использования цвета

- **Тёмная тема (основная):** фон `#3D3A38`, акцент `#7BA5B8`, текст `#F2EDE4`
- **Светлая тема (альтернативная):** фон `#F2EDE4`, акцент `#3D3A38`, детали `#C4956A`
- **Красный `#8B1A1A`** — используется редко, только для важных CTA или завершающих элементов (по аналогии с финальными страницами брендбука)
- **Золотой `#C4956A`** — тёплые акценты, ховер-состояния, иллюстративные элементы

## 3. Типографика

### 3.1 Логотипный шрифт (рукописный / руны)

- Используется **только** для написания слова `piligrim` в логотипе и декоративных заголовках
- Характер: рукописный, угловатый, напоминающий древние руны
- **Не использовать** для UI-текста и интерфейсных элементов
- В интерфейсе воспроизводится как SVG-логотип или изображение

### 3.2 Основной UI-шрифт

**Museo Sans** — главный шрифт интерфейса

```
Font Family: Museo Sans
Weights: 300 (light), 700 (bold)
Поддержка: Latin, Cyrillic (RU), Kazakh (KZ — ӘІҢҒҮҰҚӨҺ)
```

| Стиль | Вес | Размер | Применение |
|---|---|---|---|
| Display | 700 | 32–40px | Главные заголовки экранов |
| Title | 700 | 24px | Заголовки секций |
| Heading | 700 | 18–20px | Названия блюд, разделов |
| Body | 300 | 15–16px | Основной текст, описания |
| Caption | 300 | 12–13px | Подписи, метки, вторичный текст |
| Button | 700 | 14–16px | Текст кнопок (uppercase при необходимости) |

### 3.3 Правила типографики

- Межстрочный интервал (line-height): `1.5–1.7` для body, `1.2` для заголовков
- Трекинг (letter-spacing): `0.05–0.1em` для кнопок и лейблов
- Казахский и русский тексты — одинаковые стили, шрифт поддерживает оба
- **Никакого serif** — бренд полностью sans-serif в UI

## 4. Логотип

### Состав логотипа

```
[Звезда (четырёхконечная)]
        |
        | (вертикальная линия)
        |
        • (точка)
   piligrim  ← рукописный шрифт
[подпись — tagline]
```

### Вариации логотипа

| Версия | Tagline | Фон |
|---|---|---|
| Казахская | дәстүрдің дәмі мен еркіндік лебі | Голубой / тёмный |
| Русская | кухня свободы и традиций | Тёмный |
| Английская | spirit and culture cuisine | Светлый |
| Без tagline | — | Любой |
| Только звезда | — | Иконка приложения |

### Правила использования логотипа в приложении

- Логотип — всегда SVG, не растровый
- На тёмном фоне: логотип `#F2EDE4` (кремовый)
- На светлом фоне: логотип `#3D3A38` (тёмный)
- На голубом фоне (`#7BA5B8`): логотип белый `#FFFFFF`
- **Никаких кавычек**, никакого строчного написания `piligrim` в UI-текстах (только в логотипе)
- В тексте всегда: `PILIGRIM` или `Piligrim`

### Иконка приложения

- Символ звезды (четырёхконечный крест) на тёмном `#3D3A38` или голубом `#7BA5B8` фоне
- Без текста

## 5. Иконография — Знаки-тотемы

Бренд имеет собственную систему пиктограмм в стиле древних кочевнических символов.

**Характер иконок:**
- Органические, слегка неровные формы (не идеально симметричные)
- Вдохновлены казахской мифологией и петроглифами
- Цвет иконок: монохромные, в цвете акцента (`#7BA5B8`) или текста

**Использование в приложении:**
- Иконки разделов меню / навигации — используем тотемы вместо стандартных иконок
- Иллюстрации к блюдам, разделам, онбордингу
- Анимации при переходах (иконки оживают как часть путешествия)

**Доступные символы (из брендбука):** птица, луна, колесо, растения, животные, звезда — всего ~16 символов в наборе.

## 6. UI-компоненты и стиль

### 6.1 Общий визуальный характер

- **Стиль:** минималистичный, органический, с историческими отсылками
- **Ощущение:** тактильное, тёплое, немного мистическое
- **Не:** яркий, неоновый, стерильный, корпоративный
- **Геометрия:** предпочтение плавным формам, мягким скруглениям, отказ от острых прямых углов
- **Текстуры:** тёмный фон может иметь лёгкую текстуру (имитация кожи, ткани, дерева)

### 6.2 Карточки и контейнеры

```
Border-radius: 12–16px (карточки), 8px (маленькие элементы)
Background (тёмная тема): #2A2826 или rgba(255,255,255,0.05)
Border: 1px solid rgba(242,237,228,0.1)
Shadow: 0 4px 24px rgba(0,0,0,0.3)
```

### 6.3 Кнопки

**Primary Button**
```
Background: #7BA5B8 (Мөлдір су)
Text: #F2EDE4
Border-radius: 8px
Font: Museo Sans 700
Padding: 14px 24px
```

**Secondary Button**
```
Background: transparent
Border: 1.5px solid #7BA5B8
Text: #7BA5B8
```

**CTA / Action Button (финальное действие)**
```
Background: #8B1A1A (Піскен жеміс)
Text: #F2EDE4
```

**Hover / Pressed State:** затемнение на 10–15%, плавный переход `transition: 0.2s ease`

### 6.4 Навигация

- Bottom Navigation Bar — тёмный фон `#2A2826`, иконки-тотемы
- Активный таб: акцент `#7BA5B8`
- Неактивный: `rgba(242,237,228,0.4)`
- Отсутствует стандартный tab bar — используем тотем-иконки вместо стандартных

### 6.5 Разделители и пространство

```
Spacing unit: 8px (базовая сетка)
Section padding: 24px
Card gap: 16px
Divider: 1px solid rgba(242,237,228,0.1)
```

## 7. Анимации и взаимодействие

Концепция путешествия требует осмысленных переходов.

| Элемент | Анимация |
|---|---|
| Онбординг | Вертикальный скролл, звезда "ведёт" вниз по линии пути |
| Открытие блюда | Плавное появление снизу (slide-up), затухание фона |
| Переключение разделов | Cross-fade, без слайдов |
| Кнопки | Scale 0.97 при нажатии, `ease-out 0.15s` |
| Загрузка | Анимация звезды-тотема, пульсация |
| Успех / Завершение | Иконка асыка или звезды с появлением |

**Общее правило:** Анимации — плавные, не агрессивные, скорость `200–400ms`. Ощущение медитативности.

## 8. Тон и язык интерфейса

### Обращение к пользователям

- Всегда на **вы** (в переписке — со строчной буквы: "вы")
- Обращение: **герой / герои** (героиня / героини)
- Пример: _"Дорогие герои, добро пожаловать в PILIGRIM!"_

### Приветствия

- ✅ "Добрый день", "Добрый вечер", "Здравствуйте"
- ❌ "Доброго времени суток!", "Привет, друг!"

### Прощания

- ✅ "До свидания", "Хорошего дня", "До встречи", "Удачного вам пути"
- ❌ "Пока!", "Прощайте!"

### Правила написания

- Название всегда латиницей: `PILIGRIM` или `Piligrim` — **никогда** `piligrim` в тексте
- Без кавычек вокруг названия
- Без эмодзи в коммуникации
- Эмоции передаются через слова, не символы

### Тональность

| Да | Нет |
|---|---|
| Эмоциональный | Сухой |
| Остроумный | Язвительный |
| Открытый | Скрытный |
| Современный | Архаичный |
| Поэтичный | Пафосный |
| Конкретный | Многословный |

### Ключевые слова бренда (для UX-копирайтинга)

`иммерсивность` · `идентичность` · `аутентичный` · `свобода` · `традиции` · `путь` · `код` · `нарратив` · `гастрономия` · `память` · `эфемерность` · `центральноазиатский`

## 10. Фотостиль и изображения

- **Атмосфера:** тёплая, тёмная, кинематографичная, с живым светом (свечи, огонь, золотые блики)
- **Текстуры:** дерево, кожа, камень, ткань, глина
- **Цветовая обработка:** тёплые тени, мягкий контраст, никакого холодного белого
- **Ракурс:** крупные планы деталей, атмосферные кадры интерьера
- **Люди:** крупный план рук, жестов — без явных лиц или обезличенно
- **Запрещено:** стоковые фото в стиле "happy food", яркие белые фоны, неоновые цвета

---

## 11. Архетип бренда

**50% Маг/Волшебник** — мечтательность, связь с природой, загадочность, мистические символы  
**50% Герой/Воин** — трансформация, сила, преодоление, вдохновение

→ В приложении это выражается через **нарративный UX**: каждый шаг пользователя — часть истории, не просто функция.

## 12. Чеклист перед релизом дизайна

- [ ] Название бренда написано как `PILIGRIM` или `Piligrim` (не в кавычках)
- [ ] Гость называется "герой", не "клиент" / "пользователь"
- [ ] Все цвета из палитры бренда (нет посторонних)
- [ ] Шрифт Museo Sans 300/700, поддержка KZ и RU
- [ ] Логотип — SVG, адаптирован под тёмный и светлый фон
- [ ] Иконки — тотем-стиль, не стандартные Material/SF Symbols
- [ ] Тон текста — поэтичный, современный, без сухости и пафоса
- [ ] Анимации — плавные, медитативные, не быстрее 200ms
- [ ] Нет эмодзи в коммуникационных текстах
- [ ] Фото — атмосферные, тёплые, кинематографичные

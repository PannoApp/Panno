Work as a senior designer: every change must be premium, atmospheric, and precise.

**Why:** User explicitly said "как senior designer не должен ошибаться и внимательно должен делать красиво премиально чтобы все были в восторге"

**How to apply:**
- Never use Material icons where a totem SVG or custom solution fits
- Never use `CircularProgressIndicator` — create totem-based loaders
- Every empty state, loading state, and transition must feel branded
- Test spacing, colors, and animations against piligrim_design_spec.md before finalizing
- Follow the PILIGRIM_DESIGN_PLAN.md task order — one task at a time, mark done before moving on
